# reactQueens

The famous N-Queens puzzle build with [React][react] and [Tau Prolog][tau].

-   [8 Queens puzzle][8q]
-   [SWI-Prolog][swi]

### Installation

For the UI

```sh
$ npm install
$ npm run start
```

Then start the server:

```sh
$ npm run watch-server
```

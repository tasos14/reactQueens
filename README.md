# reactQueens

The famous N-Queens puzzle build with [React][react], [Redux][redux] and SWI-Prolog Pengines.

-   [8 Queens puzzle][8q]
-   [SWI-Prolog][swi]
-   [Pengines][pengines]

### Installation

For the UI

```sh
$ npm install
$ npm run start
```

Then start the Pengine server:

```sh
$ cd pengines
$ swipl run.pl
```
